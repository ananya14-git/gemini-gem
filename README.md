# 🤖 DevOps Assistant - Custom Gemini Gem

## Overview
A custom AI assistant built using **Google Gemini Gems** that helps DevOps engineers with:
- Kubernetes YAML generation
- Dockerfile creation
- CI/CD pipeline configuration (Jenkins, GitHub Actions)
- Error debugging
- Linux commands and Git workflows

## How It Works
This is a **Gemini Gem** - a custom version of Gemini AI configured with specific instructions to act as a Senior DevOps Engineer.

## Instructions Used
The gem follows these core instructions:
- **Role**: Senior DevOps Engineer
- **Response format**: TL;DR → Explanation → Solution → Verification → Pro Tip
- **Specializes in**: K8s, Docker, Jenkins, GitHub Actions, Linux/Git
